# 추가 후보 기능

## 조도센서 기반 수면모드 제안

상태: 후보

목표:

- 방이 어두워졌을 때 수면모드를 자동으로 켜지 않고, 사용자에게 활성화 여부를 묻는다.
- 불을 잠깐 끈 상황이나 영화 시청 같은 예외를 고려해 자동 전환 대신 제안 방식으로 동작한다.

예상 동작:

```text
밤 시간대
+ 재실 상태
+ 조도 낮음
+ 일정 시간 이상 어두움 유지
+ 수면모드 OFF

→ "수면모드로 전환할까요?" 표시
→ 사용자가 승인하면 수면모드 ON
```

추천 조건 초안:

- 시간대: 22:00~03:00
- 어두움 유지 시간: 10분 이상
- 재실 감지: ON
- 수면모드: OFF 상태

추천 부품:

- BH1750 조도센서: lux 단위 측정, I2C 사용, 비교적 정확함
- CdS 조도센서 모듈: 저렴하고 단순하지만 값이 대략적

우선 추천:

- BH1750

웹 화면 후보:

- 현재 밝기 표시
- 방 상태 표시: 밝음 / 어두움
- 수면모드 제안 알림
- 버튼: 수면모드 ON / 무시
